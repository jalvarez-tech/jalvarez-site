# Byte theme

Editorial-developer theme for jalvarez.tech.

- **Tokens & guidance:** see `../../../../docs/DESIGN.md`.
- **Sources:** `scss/` (CSS), `scripts/` (build helpers), `icons.manifest.json` (Lucide icons to bundle).
- **Built artefacts:** `dist/css/main.css`, `icons.svg` — gitignored, generated by CI.

## Local development

```bash
cd web/themes/custom/byte
npm install
npm run dev      # watch SCSS → dist/css/main.css
# in another terminal:
npm run build:icons
```

Then in DDEV: `ddev drush cr` and refresh the browser.

## Production build

CI runs this automatically on push to `main`:

```bash
npm ci
npm run build    # css + sprite
```

Output:
- `dist/css/main.css`
- `icons.svg`

Both are shipped to Hostinger as part of the deploy artefact.
